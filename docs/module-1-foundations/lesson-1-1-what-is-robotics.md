## What is a Robot?

A **robot** is a machine that can:

1. **Sense** its environment (using cameras, sensors, etc.)
2. **Think** about what to do (using computer programs)
3. **Act** on the world (using motors, arms, wheels, etc.)

**Key idea:** Robots follow instructions (programs) to accomplish tasks **automatically**â€”without human control for every action.

### Robot vs. Machine

Not all machines are robots:

| Machine | Robot |
|---------|-------|
| A toaster heats bread | A robot arm assembles cars |
| A vacuum cleaner has one job | A robot vacuum maps your home |
| Does exactly what you tell it | Learns and adapts to changes |

## Real-World Examples

### 1. **Boston Dynamics Spot**
A four-legged robot that walks like a dog. It:
- Senses terrain with cameras and lidar
- Processes movement commands
- Controls four motors to walk and climb

### 2. **Amazon Warehouse Robots**
Orange robots that move shelves to workers:
- Sense shelf locations
- Navigate around obstacles
- Pick and sort items

### 3. **Self-Driving Cars**
Robots on wheels:
- See the road with cameras and radar
- Process traffic, pedestrians, signs
- Control steering, acceleration, braking

### 4. **Robotic Arms in Factories**
Industrial robots that weld and assemble:
- Sense part positions
- Execute precise movements
- Never get tired

## Key Concepts

| Term | Definition |
|------|-----------|
| **Autonomous** | Works without continuous human control |
| **Sensor** | Device that detects changes in the environment |
| **Actuator** | Device that creates motion or action |
| **Algorithm** | Step-by-step instructions for solving a problem |
| **AI (Artificial Intelligence)** | Computer program that learns and makes decisions |
